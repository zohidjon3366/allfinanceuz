ALL FINANCE — JAMOA MA’LUMOTLARI YANGILANGAN YAKUNIY VERSIYA

1. GitHub repository ichidagi eski fayllarni ushbu paketdagi fayllar bilan to‘liq almashtiring.
2. assets/team papkasini ham to‘liq yuklang.
3. Render turi: Web Service
   Build Command: npm install
   Start Command: npm start
4. Environment Variables:
   TELEGRAM_BOT_TOKEN=...
   TELEGRAM_CHAT_ID=6279154555
5. Deploy tugagach brauzerda Ctrl + F5 bosing.

Yangilanishlar:
- barcha 7 nafar jamoa a’zosining to‘liq F.I.Sh., lavozimi va tajribasi kiritildi;
- jamoa ma’lumotlari bosh sahifada ham ko‘rinadi;
- jamoa sahifasidagi kartalar va tajriba belgilarining dizayni yaxshilandi.


Qo'shimcha yangilanishlar:
- yuqori menyu orqasiga xira O'zbekiston bayrog'i joylashtirildi;
- xizmat sahifalaridagi 4 bosqichli rasm kichraytirildi;
- manzil bo'limidagi Google Maps ko'rinishi ochiqroq qilindi.
