ALL FINANCE — YAKUNIY TUZATILGAN PREMIUM VERSIYA

1. GitHub repository ichidagi eski fayllarni to‘liq almashtiring.
2. Ushbu paketdagi barcha fayl va papkalarni repository ildiziga yuklang.
3. Render turi: Web Service.
4. Build Command: npm install
5. Start Command: npm start
6. Environment Variables:
   TELEGRAM_BOT_TOKEN=...
   TELEGRAM_CHAT_ID=6279154555
7. Deploy tugagach Ctrl + F5 bosing.

Tuzatishlar:
- barcha 8 ta xizmat sahifasidagi “4 bosqichli yondashuv” bo‘limi premium rasm va ixcham bosqich kartalari bilan yangilandi;
