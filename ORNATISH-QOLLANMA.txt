ALL FINANCE — ishlaydigan yakuniy versiya

1. GitHub repository ichidagi eski fayllarni TO‘LIQ o‘chirib, shu paketdagi barcha fayllarni yuklang.
2. Ayniqsa assets papkasi to‘liq yangilanishi shart.
3. Render turi: Web Service.
4. Build Command: npm install
5. Start Command: npm start
6. Environment: TELEGRAM_BOT_TOKEN va TELEGRAM_CHAT_ID ni saqlang.
7. Deploy tugagach brauzerda Ctrl + F5 bosing.

Eslatma: ushbu versiyada xizmat ikonkalari inline SVG ko‘rinishiga o‘tkazilgan, shuning uchun rasmlar tushmay qolish muammosi bartaraf etilgan. Manzil kartasi orqasida Toshkent xaritasi uslubidagi lokal SVG fon qo‘shilgan.
