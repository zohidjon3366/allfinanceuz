# ALL FINANCE — GitHub Pages sayti

Bu repozitoriy ALL FINANCE kompaniyasining statik veb-sayti uchun tayyorlangan.

## GitHub orqali joylashtirish

1. GitHub'da `allfinance-site` nomli yangi **Public** repository yarating.
2. Ushbu papkadagi barcha fayllarni repository ildiziga yuklang. `index.html` repozitoriy ichida darhol ko‘rinishi kerak; uni yana bir papka ichiga joylamang.
3. Repository sahifasida **Settings → Pages** bo‘limiga o‘ting.
4. **Build and deployment → Source** qismida **Deploy from a branch** ni tanlang.
5. **Branch:** `main`, **Folder:** `/(root)` ni tanlab **Save** bosing.
6. Bir necha daqiqadan keyin sayt quyidagi manzilda ochiladi:

   `https://USERNAME.github.io/allfinance-site/`

7. `TILDA-IFRAME-KODI-GITHUB.txt` ichida `USERNAME` va `REPOSITORY` ni o‘zingiznikiga almashtiring va kodni Tilda T123 blokiga joylang.

## Keyingi yangilashlar

GitHub repository ichidagi faylni o‘zgartirib `Commit changes` qilsangiz, GitHub Pages saytni avtomatik qayta nashr qiladi.

## Muhim

- GitHub Pages — statik hosting. HTML, CSS va JavaScript ishlaydi.
- Maxfiy parollar, API kalitlari va server ma’lumotlarini repozitoriyga yozmang.
- Konsultatsiya shakli hozir Telegram ulashish oynasini ochadi; serverga ma’lumot saqlamaydi.
- `allfinance.uz` domenini keyin GitHub Pages'ga to‘g‘ridan-to‘g‘ri ulash mumkin.
